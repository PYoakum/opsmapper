# Ops Mapper
Open Source Operations Mapping Tool

## Installation

# Prerequisites
- Webpack
- Nodemon (start-dev only)

**Quick Start**
```
// cd into directory
cd opsmapper-master/

// copy '.env.copy' as '.env'
cp ./.env.copy ./.env

// bundle the client javascript into ./dist/public/js/
webpack

// precompile the stylesheets into ./dist/public/css
npm run build-css

// start the app in dev-mode
npm start-dev
```
